# 경제뉴스 요약 프롬프트

한국 경제뉴스 기사를 출근 전 빠르게 훑을 수 있도록 요약. 원문이 한국어라 번역 필요 없음 — 핵심 내용을 압축하는 게 목표.

## 반드시 포함할 필드

각 항목에 대해 JSON 으로 반환:

- `id`: 입력과 동일
- `summary`: 2~3줄 한국어 요약

## summary 작성 규칙 — 엄격히 준수

### 원칙 1: 제목 재진술 금지

❌ 나쁨 (제목만 바꿔 쓴 수준):
> 원달러 환율이 1,380원을 돌파해 3개월 만에 최고치를 기록했다.

✅ 좋음 (lead 의 구체 수치·원인·영향 포함):
> 원달러 환율이 전일 대비 6.2원 오른 1,381.4원에 마감해 3개월 만에 최고치. 미 고용지표 호조로 달러 강세가 재개된 영향. 외국인 자금 유출 우려가 커진다.

### 원칙 2: lead 에서 수치·인물·기관명·기간 반드시 추출

lead 문단에 다음이 있으면 **무조건** summary 에 포함:
- 수치: `3.50%`, `1,381.4원`, `전년 대비 +18.3%`, `2조 원`
- 고유명사: 한국은행, 연준, 파월, 삼성전자, TSMC 등 기관·인물·기업명
- 이벤트 시점: `4월 금통위`, `3월 수출`, `올해 말`
- 원인-결과 고리: `미 고용 호조 → 달러 강세` 같은 인과

### 원칙 3: lead 가 빈약할 때

- 제목이 가진 정보만으로 정확히 풀어서 서술. **수치 모르면 수치 언급하지 말 것**.
- 추측·전망 금지: "전망이다", "예상된다", "보인다" 같은 표현 제거.

### 원칙 4: 금지 어휘

- "업계에서는", "시장에서는" 같은 출처 불명 인용 → 삭제
- "전망이다", "예상된다" → 현재형으로 변경하거나 삭제
- 이모지, 강조 기호(`**`, `*`) 절대 금지

### 원칙 5: 길이

- 1~2문장: 무엇이 일어났는가 (사건) · 핵심 수치·주체 포함
- 3문장 허용: 왜 중요한가 / 배경 / 파급 효과 — lead 근거가 있을 때만
- 총 60~160자 범위

## 입력 형식

```json
{
  "items": [
    {
      "id": "item_0",
      "title": "원달러 환율 1,380원 돌파",
      "lead": "17일 서울 외환시장에서 원달러 환율이 전일 대비 6.2원 오른 1,381.4원에 마감했다. 미국 고용지표 호조로 달러 강세가 재개된 영향...",
      "source": "한국경제"
    }
  ]
}
```

## 출력 형식 (JSON 만, 다른 설명 금지)

```json
{
  "summaries": [
    {
      "id": "item_0",
      "title_ko": "",
      "summary": "원달러 환율이 전일 대비 6.2원 오른 1,381.4원에 마감, 3개월 만에 최고치. 미 고용지표 호조로 달러 강세 재개가 배경. 외국인 자금 유출 우려가 확대된다."
    }
  ]
}
```

`title_ko` 는 원문 제목이 한국어이므로 **빈 문자열** 로 남긴다.
